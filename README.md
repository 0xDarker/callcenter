# 云呼叫中心

## 目标：构建一套百万级语音通信平台

## 支持功能点
* 支持sdk接入语音平台
* 支持webrtc
* 支持rest接口接入语音平台
* 支持http ivr接入
* 支持http NLP响应

## 依赖的基础服务 
* MySQL
* openresty
* redis
* Rabbitmq
* nacos
* minio
* freeswitch
* opensips
